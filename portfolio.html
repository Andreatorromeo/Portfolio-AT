The idea

I wanted to create my first portfolio website comprehensive of my main projects made during the master at the Scuola Politecnica di Design.

The codes

After designing it with Adobe XD, I used HTML, CSS, JS for the creation of this website, currently it's hosted by Netlify and Github.

Pain points

  • Creating a design with Adobe XD which i can create with codes too
  • Reproducing with codes the same design
  • Adapting the design to different screen sizes

Strategy

  • Creating a clear and consistent design
  • Having the same HTML structure among different projects
  • Creating a homepage where you can find all the info about me

Homepage

The homepage is composed by four sections which are linked in the navbarwith relative links. I wanted to create a ripetitive scheme of the background simple and clean using the meta-elements.

Projects

I wanted to have the same HTML structure for projects pages. The challenge at this point was to create a scheme which was good for all my projects without breaking any flow

GitHub and Visual studio

For this project I learned to use GitHub for the creation of repositories, comunicate with the teacher, merging branches, opening issues etc. As software for the codes, I used Visual Studio Code.
